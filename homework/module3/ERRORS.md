# Ошибки в классе Person и их исправления

## 1. Ошибка в методе `get_age()`
- **Проблема**: `return self.yob - now.year` — вычитание года рождения из текущего года даёт отрицательный возраст (если год рождения меньше текущего).
- **Исправление**: `return now.year - self.yob`

## 2. Ошибка в методе `set_name()`
- **Проблема**: `self.name = self.name` — присваивание самому себе, параметр `name` не используется.
- **Исправление**: `self.name = name`

## 3. Ошибка в методе `set_address()`
- **Проблема**: `self.address == address` — использовано сравнение (`==`), а не присваивание (`=`).
- **Исправление**: `self.address = address`

## 4. Ошибка в методе `is_homeless()`
- **Проблема**: `return address is None` — переменная `address` не определена. Следует обращаться к атрибуту экземпляра `self.address`.
- **Исправление**: `return self.address == ''` (или `return not self.address`). В условии сказано: "returns True if address is not set". В конструкторе по умолчанию `address = ''`, значит, отсутствие адреса — это пустая строка, а не `None`.

## 5. Отсутствие импорта datetime
- **Проблема**: в классе используется `datetime.datetime.now()`, но модуль `datetime` не импортирован.
- **Исправление**: добавить `import datetime` в начало файла.
